\# 🍔 Food Delivery Web Application



A full-stack Food Delivery Web Application developed using Java, Servlets, JSP, JDBC, MySQL, HTML, CSS, and JavaScript.



\## 🚀 Features

\- User Registration \& Login

\- Restaurant Listing

\- Menu Display

\- Add to Cart

\- Checkout

\- MySQL Database Integration

\- Session Management



\## 🛠 Technologies Used

\- Java

\- Servlets

\- JSP

\- JDBC

\- MySQL

\- HTML

\- CSS

\- JavaScript



\## 🗄 Database

MySQL



\## ▶️ How to Run

1\. Import the project into Eclipse.

2\. Configure Apache Tomcat.

3\. Create the MySQL database.

4\. Update the database credentials in `DBConnection.java`.

5\. Run the project on the Tomcat server.
